## React.js nedir?

React, Facebook tarafından geliştirilen ve kullanıcı arayüzü oluşturmak için kullanılan bir JavaScript kütüphanesidir.

## Örnek kullanım

```jsx title="App.jsx"
import React from 'react';

export default function App() {
  return (
    <div>
      <h1>Merhaba Dünya</h1>
    </div>
  );
}
```
