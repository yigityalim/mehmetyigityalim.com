# Bootstrap

## Nedir?

Bootstrap, Twitter tarafından geliştirilen ve açık kaynak kodlu bir CSS framework'üdür. Bootstrap, web siteleri ve web uygulamaları geliştirmek için kullanılan en popüler CSS framework'lerinden biridir. Bootstrap, HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

## Neden Kullanılır?

Bootstrap, web sitelerinin ve web uygulamalarının tasarımını hızlı ve kolay bir şekilde yapmak için kullanılır. Bootstrap, web sitelerinin ve web uygulamalarının tasarımını yaparken, responsive, mobile-first ve modern tasarım prensiplerini göz önünde bulundurarak tasarım yapmanıza olanak sağlar.

-   **Responsive Tasarım**: Bootstrap, responsive tasarım prensiplerini göz önünde bulundurarak tasarım yapmanıza olanak sağlar. Bu sayede, web siteleriniz ve web uygulamalarınız, farklı ekran boyutlarına uyumlu bir şekilde görüntülenebilir.
-   **Mobile-First Tasarım**: Bootstrap, mobile-first tasarım prensiplerini göz önünde bulundurarak tasarım yapmanıza olanak sağlar. Bu sayede, web siteleriniz ve web uygulamalarınız, mobil cihazlara uyumlu bir şekilde görüntülenebilir.
-   **Modern Tasarım**: Bootstrap, modern tasarım prensiplerini göz önünde bulundurarak tasarım yapmanıza olanak sağlar. Bu sayede, web siteleriniz ve web uygulamalarınız, modern bir tasarıma sahip olabilir.
-   **Hızlı ve Kolay Tasarım**: Bootstrap, web sitelerinin ve web uygulamalarının tasarımını hızlı ve kolay bir şekilde yapmanıza olanak sağlar. Bu sayede, web siteleriniz ve web uygulamalarınız, kısa sürede tasarlanabilir.

## Nasıl Kullanılır?

<Steps>
### Adım 1: Bootstrap cdn'ini html kodunuza ekleyin

```html
<!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet" />
        <title>Document</title>
    </head>
    <body>
        <h1>Hello, world!</h1>
        <script src="https://cdn.j
    sdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
    </body>
</html>
```

1. Not: Bootstrap 5.3.0 sürümünü kullanıyoruz. Sürümü değiştirmek isterseniz, `<link>` ve `<script>` etiketlerindeki sürüm numarasını değiştirebilirsiniz.
2. Not: Eğer bootstrap javascript dosyalarını kullanmak istemiyorsanız, `<script>` etiketini kaldırabilirsiniz.
3. Not: Bootstrap cdn'ini eklediğiniz `html` dosyası, sizin index.html yani ana sayfanız olmalıdır. Diğer sayfalarda bootstrap cdn'ini eklemek istiyorsanız, o sayfaların da `html` dosyalarına bootstrap cdn'ini eklemelisiniz.

</Steps>
