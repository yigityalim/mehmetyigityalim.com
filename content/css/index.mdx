## CSS Index

```css title="style.css" {2, 6}
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

h1:hover {
  color: red;
}
```

### CSS Syntax

```css
selector {
  property: value;
}
```

### CSS Selectors

> Selectors are used to "find" (or select) the HTML elements you want to style.

- **Universal Selector** - `*`
- **Type Selector** - `h1`
- **Class Selector** - `.class`
- **ID Selector** - `#id`
- **Attribute Selector** - `[attribute=value]`
- **Pseudo-class Selector** - `:hover`
- **Pseudo-element Selector** - `::first-line`
